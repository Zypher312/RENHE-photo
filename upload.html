投稿/上传

欢迎提交照片。所有投稿都会先进入审核队列，通过后才会展示在画廊。

投稿规则

  • 仅接受你拥有版权或已获授权的照片。
  • 建议 JPG/PNG/WebP，单张不超过50MB。
  • 发布前我可能会压缩尺寸并清理 EXIF。
  • 注册Google账号可能会一时有一点点麻烦，但是有助于编辑表单也大大帮开发者节约了时间，希望有条件的产出姐姐们可以注册一下QAQ。
  • 填写这个表单应该需要梯子...内地网速过于阻塞QAQ。

打开上传表单 →

如果没有Google账号请点击此链接上传 →

建议你填写这些信息（可写在文件夹/文件名/备注里）

  • 图片上传者：你的名字 / 昵称
  • 拍摄时间：YYYY-MM-DD
  • 涉及人物：例如 A / B / 无
  • 内容类型：比赛实况 / 训练物料 / 路透 / 饭制同人

小提示：上传网盘链接的朋友你可以用文件名格式，例如：2026-01-30_上传者_内容类型_人物.jpg
